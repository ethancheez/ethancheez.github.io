Cheezy Escape was a side project to explore the Java programming language and to improve my skills in game development. This platform game consisted of unique levels, with a final boss battle at the end. The goal in each level is to advance to the door that brings you to the next level. There were three classes to choose from (knight, ninja, and wizard), each with their own special abilities that can help you in beating the game. 

I worked on this project for a few months and went through multiple versions after receiving feedback from different people who tested my game. I began off creating the display with only one level and user input. Then I added temporary platforms to test the moving and jumping mechanics of the player. Once the controls were set, I added sprites and textures to the display then began creating the levels. Upon completion of all four levels of the game, I created a menu system that allowed the player to understand how to play and navigate through the different levels. Lastly, I added in multiple character classes, each with their special moves. The final touch of the game was adding audio into the game.

I learned a lot about the Java programming language because this project contained a lot of classes and libraries. For example, it was the first time I used JFrame, JButtons, and keyboard inputs. I had to teach myself these concepts because they were not part of the computer science curriculum. In the end, I was able to understand most of how these worked that allowed me to complete the game and also teach my peer programmers about JFrame.

The source code of the game can be found in this [GitHub Repo](https://github.com/ethancheez/cheezy-escape)
