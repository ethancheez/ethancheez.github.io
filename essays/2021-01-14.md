Nobody is ever skilled at anything in the beginning; it takes continuous practice and dedication to truly master a subject. To be honest, I never thought I would be interested in programming because I had a deep interest in electricity and circuits since the beginning of my high school years. It was not until I realized that a lot of software was required in various electrical components that got me interested in programming. I was a novice programmer going into the computer science field during my senior year of high school, starting off with programming in Java. It was almost like learning a new language because of the specific syntax I had to understand. Eventually, I began to settle into programming and grasp the concepts of syntax and program design, which allowed me to improve my skills in Java as well as incorporating them into other languages like C and C++. With interest in both electrical engineering and computer science, I have chosen the path of computer engineering where I will master my skills in both hardware and software engineering.

My greatest interests in software engineering involve understanding artificial intelligence, web development, and app development such as games. I have done robotics in the past that involved autonomous code, but it did not have any AI. I never fully understood how a body of text could act like a brain that reacts to its environment. I have seen this in a tangible object, like Boston Dynamics' robot dog, or in a video game that contains a computer player, or a "bot". These things made me really interested in learning software development for myself to see if I could recreate these in the future. Web development is another part of software engineering because we all use computers and the internet on a daily basis. I am interested in understanding this virtual network we live in and contribute more into it, whether it is simply creating a website or server, or as difficult as working with network control systems. I think software engineering has many disciplines and I would like to explore the possibilities through this class.

<img class="ui medium right floated image" src="../images/programming.jpg">

Software engineering will most likely be difficult and I would like to gain the skills necessary for success. Such skills may include efficient programming or dynamic programming, which will allow me to spend less time programming, reduce the runtime, and make it easier to navigate through code. I would like to think of creative designs for software, whether if it is a game design, app design, or website design. By gaining experience with the different software development projects in this class, I will be able to feel more comfortable with software engineering in the future.
