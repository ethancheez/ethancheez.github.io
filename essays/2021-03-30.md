## Overview
*The problem*: Lonely UH gamers throughout the campus. We need a break from the workload of classes. Rather than playing games by yourself, they should be able to connect with other gamers at UH!

*The solution*: A hub to link with other UH gamers and organize a team to play together. Or just cry together. This will make it much easier to organize LAN parties, tournaments, or any form of competitive play. The hub will group UH gamers that will make it easier to connect with others with the same game interests. 

## Mockup Page Ideas
1. Home page
2. Sign up page (to initialize game interests for said user)
3. Game directory page to separate game interests
4. Within the game directory page, there will be a list of all UH gamers with the same interest and their contact information.
5. Admin page to display the entire list of users with their game interests.
