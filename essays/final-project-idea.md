## Overview
*The problem*: Lonely UH gamers throughout the campus. We need a break from the workload of classes. Rather than playing games by yourself, they should be able to connect with other gamers at UH!

*The solution*: A hub to link with other UH gamers and organize a team to play together. Or just cry together. This will make it much easier to organize LAN parties, tournaments, or any form of competitive play. The hub will group UH gamers that will make it easier to connect with others with the same game interests. 

## Mockup Page Ideas
1. Landing page with list of game interests
2. Sign up page
3. Game directory page that allows you to join one or more groups with a specific game interest.
4. Within the game directory page, there will be a list of all UH gamers with the same interest and their contact information.
5. Admin page to display the entire list of users with their game interests.
6. Discussion Thread for each category
7. Suggestion page to request a specific game interest to be added.

## Use Case Ideas
Whether or not the following bullet points list all pages or not, the completed use case should show an end-to-end scenario of using the system.

1. New user goes to landing page, looks up the list of game interests, signs up for an account with contact information and currently played games.
2. User goes to landing page, signs in to account, go to game directory page to join groups with their game interests, clicks through each group to find a list of all other UH gamers with the same game interests, contact them using their contact information.
3. User goes to landing page, signs in to account, go to game directory page to leave groups they no longer want to be a part of.
4. Admin goes to landing page, logs in, adds a new category of game interests.
